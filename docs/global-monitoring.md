# Alora Global Monitoring 2.0

## 🌍 Global Observability
- **Latency Heatmap**: Real-time view of API latency by country.
- **Failover Status**: Status of Multi-Region replicas.

## 💰 Business Metrics
- **ARR (Annual Recurring Revenue)**: Real-time via Lemon Squeezy webhook.
- **HeartTokens Velocity**: Rate of earn vs burn.

## 🚨 Predictive Alerts
- **Anomaly Detection**: IF login_failures > 2 * baseline THEN Alert.
- **Fraud Spike**: IF new_accounts > 500/min THEN Lockdown Mode.
