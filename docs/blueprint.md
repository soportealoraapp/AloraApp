# **App Name**: Alora: Her Safe Space

## Core Features:

- Welcome Screen: Welcome screen with Alora logo and inspirational message.
- Registration/Onboarding: Registration/Onboarding with email, Google, or Apple; obligatory selfie verification; questions about interests and lifestyle.
- Profile Discovery: Home screen displays profiles with photos, key information, and interest compatibility.
- Safe Chat: Enables chat function initiation only by women. Profiles have visible report/block buttons.
- AI Anti-Offensive Filter: Filter for offensive content: the tool scans incoming messages for potentially offensive language, and automatically filters them out before they reach the recipient.
- Compatibility Section: Section to show compatibilities and interest overlaps via short quizzes.
- Referral Program: Users can refer female friends to promote organic growth of the community. A special referral system grants priority or exclusive access features to the referring user as an incentive, enhancing community engagement and exclusivity.

## Style Guidelines:

- Primary color: Pale rose (#F8BBD0), evoking softness and femininity.
- Background color: Very light rose (#FDE7EF), for a gentle, inviting feel.
- Accent color: Light purple (#D0B0F8), creating a subtle contrast and adding a touch of elegance.
- Font: 'PT Sans', a humanist sans-serif that feels both modern and approachable. Note: currently only Google Fonts are supported.
- Use clear, non-aggressive feminine icons.
- Clean, minimal layout with generous spacing and a focus on large, high-quality profile photos.
- Subtle animations and transitions to create a smooth and engaging user experience.
